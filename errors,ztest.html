Type 1 and Type 2 Errors in Hypothesis Testing

Confusion Matrix Representation

|                 | Decision: Reject H‚ÇÄ                 | Decision: Fail to Reject H‚ÇÄ         |
|-----------------|-------------------------------------|-------------------------------------|
| **H‚ÇÄ is TRUE**  | Type 1 Error (False Positive)       | ‚úÖ Correct Decision (True Negative) |
| **H‚ÇÄ is FALSE** | ‚úÖ Correct Decision (True Positive) | Type 2 Error (False Negative)       |

Type 1 Error (False Positive)

‚úÖ Definition: Rejecting the null hypothesis (H‚ÇÄ) when it is actually TRUE.
‚úÖ Probability of Occurrence: Denoted by Œ± (Significance Level), typically set to 0.05 (5%).
‚úÖ Consequence: You detect an effect that DOES NOT exist.
Example:
  • A COVID test detects the virus in a healthy person (False Alarm).
  • Convicting an innocent person in a court trial.

Type 2 Error (False Negative)

‚úÖ Definition: Failing to reject the null hypothesis (H‚ÇÄ) when it is actually FALSE.
‚úÖ Probability of Occurrence: Denoted by Œ≤, related to statistical power (1 - Œ≤).
‚úÖ Consequence: You MISS detecting a real effect.
Example:
  • A COVID test fails to detect the virus in an infected person.
  • Letting a guilty person go free in court.

Comparison: Type 1 vs Type 2 Errors

| Error Type   |  What Happens?                                                  | Example                              | Probability            |
|--------------|-----------------------------------------------------------------|--------------------------------------|------------------------|
| Type 1 Error | Rejecting H‚ÇÄ when it is actually TRUE (False Positive)          | Finding an effect that doesn‚Äôt exist | Œ± (Significance Level) |
| Type 2 Error | Failing to reject H‚ÇÄ when it is actually FALSE (False Negative) | Missing a real effect                | Œ≤ (Related to Power)   |

How to Reduce These Errors?

üîπ Lower Type 1 Error (Œ±):
- Decrease significance level (Œ± = 0.01 instead of 0.05).
- Use larger sample sizes to improve accuracy.

üîπ Lower Type 2 Error (Œ≤):
- Increase sample size for more statistical power.
- Increase effect size (look for bigger differences).

Final Summary

- **Type 1 Error** ‚Üí False Positive (Detecting something that isn‚Äôt there).
- **Type 2 Error** ‚Üí False Negative (Missing something that actually exists).
